# Interactive Brent Oil Price Analysis Dashboard (Python & Power BI)

I developed a comprehensive data visualization tool to analyze 30 years of Brent Oil historical price trends (1996–2026). This project demonstrates the full data lifecycle: raw data extraction from exchange API, cleaning with SQL and Python, and interactive reporting in Power BI.

### Key Features

Historical price trends (1996–2026)
Monthly and annual aggregates, seasonality, volatility
Interactive filters by period, region, and indicators
Metrics comparison and report export

### Data Source

Oil exchange API → CSV
Preprocessing: SQL (cleaning/aggregation), Python (analysis, additional transformations)

### Technologies

Python (pandas, matplotlib/seaborn, Jupyter)
SQL (ETL / cleaning)
Power BI (interactive reports)
CSV (data files)
